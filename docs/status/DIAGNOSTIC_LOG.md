# 问题诊断

## 用户反馈的问题

1. 快捷键按下去没有声音提示
2. 图标没有变化
3. 黄色通知亮但没有完整输出
4. 再按后光标有问题
5. 没有看到剪切板内容

## 可能的原因

### 问题 1: 没有声音
- 声音反馈代码存在，但可能 `NSSound` 在沙盒中无法播放
- 或者 `soundFeedbackEnabled` 默认未正确设置

### 问题 2: 图标不变化 + 黄色通知亮
- 黄色 = .processing 状态
- 说明 startRecording() 成功，但处理卡住了
- 可能卡在：
  - WhisperKit 识别（首次需要下载模型）
  - DashScope API 调用（需要网络）
  - 润色服务

### 问题 3: 没有输出
- 文字注入失败
- 或者处理流程中途报错

### 问题 4: 光标问题
- 剪贴板保护机制可能有问题
- 或者 pasteChunk 在失败时没有正确结束

## 需要添加的日志

在 Pipeline 的每个阶段添加 print 日志，以便定位卡在哪里。
